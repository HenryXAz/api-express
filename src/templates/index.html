API de personas

Recursos disponibles en las siguientes rutas: 

      
    // 1. Obtener todos los registros
    
    GET /api/persons

    -----------------------------------------------------------
      
    
      
    // 2. Nuevo Registro

    POST /api/persons
    Content-Type: application/json;

    {
      "name": "nombre de la persona",
      "profession": "profesión u ocupación de la persona"
    }

    -----------------------------------------------------------
      
    
      
    // 3. Obtener Registro Específico
 
    GET /api/persons/{id_persona}
    
    -----------------------------------------------------------
      
    
      
    // 4. Actualizar Registro
    
    PUT /api/persons/{id_persona}
    Content-Type: application/json;

    {
      "name": "nuevo nombre",
      "profession": "nueva profesión u ocupación"
    }

    -----------------------------------------------------------
      
    
      
    // 5. Eliminar Registro

    DELETE /api/persons/{id_persona}
    
    -----------------------------------------------------------
